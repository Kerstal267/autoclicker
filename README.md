

# 🖱️ Auto Clicker (Python + Tkinter)
<img width="296" height="249" alt="image" src="https://github.com/user-attachments/assets/1aa6eb6f-b3d5-4452-9139-c038c3b27706" />

Простое GUI-приложение на Python для автоматического клика мышью с заданной скоростью.
Поддерживает горячие клавиши и удобный интерфейс.

---

## 📌 Описание

Этот проект представляет собой **автокликер с графическим интерфейсом**, созданный с помощью `Tkinter`.

Пользователь может:

* задать количество кликов в секунду
* запустить автокликер
* остановить его через клавишу `ESC`
* управлять приложением через GUI

---

## 🚀 Возможности

* 🖱️ Автоматический клик мыши
* ⚡ Настройка скорости (клики/сек)
* ⌨️ Горячие клавиши:

  * `ESC` — остановка и выход
  * `I` — информация о программе
* 🖼️ Удобный графический интерфейс
* 🔄 Асинхронная работа через `Tkinter.after()`


## 🖥️ Как пользоваться

1. Введи количество кликов в секунду (например: `10`)
2. Нажми кнопку **"Почати"**
3. Кликер начнёт работать
4. Нажми `ESC`, чтобы остановить и выйти

---

## ⚠️ Важно

* Программа управляет мышью — будь осторожен
* Работает глобально (не только в окне приложения)
* На некоторых системах может потребоваться запуск от имени администратора

---

## 🧠 Как это работает

Основная логика:

* Пользователь вводит `clicks_per_second`
* Программа рассчитывает задержку:

```python
delay = 1000 / clicks_per_second
```

* Используется `root.after()` для циклического выполнения кликов без блокировки GUI
* Клики выполняются через библиотеку `mouse`

---

## 📦 Используемые библиотеки

* `tkinter` — графический интерфейс
* `mouse` — управление мышью
* `keyboard` — обработка клавиш
* `time` — задержки (в данном случае почти не используется)
